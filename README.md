EcoTech Solutions - Proyecto Final JavaScript Avanzado

Descripción
Este proyecto consiste en una plataforma web corporativa para EcoTech Solutions, una consultoría tecnológica enfocada en la sostenibilidad. El objetivo principal es demostrar el dominio de la manipulación del DOM, el uso de APIs asíncronas (AJAX/Fetch), validaciones mediante expresiones regulares e integración de mapas dinámicos.

Tecnologías Utilizadas
HTML5: Estructura semántica validada por la W3C.
CSS3: Diseño responsivo, Grid, Flexbox y Navegación Sticky.
JavaScript (ES6+): Lógica funcional para cálculos, validaciones y DOM dinámico.
Leaflet.js: Integración de mapas con OpenStreetMap.
AJAX / Fetch API: Carga de noticias externas desde un archivo JSON.

Estructura del Proyecto
El proyecto sigue una organización profesional de directorios:
 /  (Raíz): Contiene el  index.html  principal y documentación.
 /CSS : Hoja de estilos global ( style.css ).
 /js : Lógica de programación ( main.js ).
 /data : Fuente de datos externa ( noticias.json ).
 /views : Páginas secundarias (Galería, Presupuesto, Contacto).
 /images : Activos visuales y fotos del proyecto.

Características Destacadas
Presupuesto Dinámico: Cálculo instantáneo de costos con aplicación de descuentos automáticos (10% en plazos > 6 meses) sin necesidad de refrescar la página.
Validación: Uso de RegEx para asegurar la integridad de datos en nombres y apellidos.
Geolocalización: El mapa permite trazar la ruta desde la ubicación actual del usuario hasta la oficina central.
Carga Asíncrona: Las noticias se actualizan mediante una petición fetch a un archivo JSON local.
